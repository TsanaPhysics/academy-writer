# Science Academic Writer (Academic Specialist AI Skill)

โครงร่างและชุดคำสั่ง (Prompt/Skill) สำหรับ AI เพื่อช่วยในการเขียนหนังสือ ตำรา และเอกสารคำสอน สำหรับขอตำแหน่งทางวิชาการระดับรองศาสตราจารย์ (รศ.) โดยเน้นด้านวิทยาศาสตร์และปัญญาประดิษฐ์ประยุกต์

## 🌟 จุดเด่น
- **รศ. Standard**: รองรับมาตรฐานการเขียนเพื่อขอตำแหน่งทางวิชาการ (ตำรา, เอกสารคำสอน)
- **Multi-Domain**: ครอบคลุมด้านฟิสิกส์, จุลชีววิทยา, AI ในการเกษตร, สิ่งแวดล้อม และการศึกษา
- **Technical Rigor**: รองรับการเขียนสมการ LaTeX และศัพท์เฉพาะทางวิทยาศาสตร์

## 📂 โครงสร้างไฟล์
- `SKILL.md`: คำสั่งหลักสำหรับ AI (Workflow)
- `REFERENCE.md`: คู่มือรูปแบบการอ้างอิงและคำศัพท์เฉพาะทาง
- `EXAMPLES.md`: ตัวอย่างการเขียนเปรียบเทียบระหว่างภาษาทั่วไปกับภาษาเชิงวิชาการ

## 🚀 วิธีใช้งานกับ AI ต่างๆ (ChatGPT, Gemini, Grok, Claude)
คุณสามารถคัดลอกเนื้อหาใน `SKILL.md` และ `REFERENCE.md` ไปวางในช่องแชทของ AI แล้วบอกว่า:
> *"ช่วยจดจำคำสั่งและแนวทางในไฟล์เหล่านี้ และใช้เป็นมาตรฐานในการช่วยฉันร่างเนื้อหาตำราวิชาการต่อจากนี้"*
