# 🎓 Science Academic Writer: Modular AI Suite (Pro Version)
**ชุดคำสั่ง AI อัจฉริยะเพื่อการผลิตผลงานวิชาการระดับสูง (ผศ., รศ., ศ.)**

ยินดีต้อนรับสู่ระบบนิเวศทางวิชาการดิจิทัล **Science Academic Writer** ชุดเครื่องมือระดับสากลที่แยกความเชี่ยวชาญออกเป็น 6 ส่วน เพื่อรองรับทั้งการเขียนตำรา (Textbooks) และบทความวิจัยนานาชาติ (Q1-Q4)

---

## 🛠️ โครงสร้างเครื่องมือ (Sub-Skills Suite)

### 1. 📖 [Academic Textbook Architect](.agents/skills/academic-textbook-architect)
- **เน้น:** การวางโครงสร้างตำรา/หนังสือ, คุมมาตรฐาน รศ., การใส่จุดประสงค์การเรียนรู้ และแบบฝึกหัด
- **เป้าหมาย:** สร้าง "กระดูกงู" ที่ลุ่มลึกและถูกต้องตามหลักครุศาสตร์

### 2. 🔬 [Science Academic Writer (Research)](.agents/skills/science-academic-writer)
- **เน้น:** การเขียนบทความวิจัย (Research Article) เพื่อตีพิมพ์ในวารสารนานาชาติ **Q1-Q4**
- **เป้าหมาย:** การเขียน IMRAD ที่แข็งแกร่ง, การจัดการ Reviewers, และการเขียน Cover Letter

### 3. 🚜 [Agro-Physicist](.agents/skills/agro-physicist)
- **เน้น:** สังเคราะห์เนื้อหาฟิสิกส์ประยุกต์, เกษตรอัจฉริยะ (IoT), คุณภาพดิน/น้ำ และโรคพืช
- **เป้าหมาย:** เนื้อหาเชิงวิชาการที่เข้มข้นและประยุกต์ใช้ได้จริง

### 4. 💻 [Scientific Coder](.agents/skills/scientific-coder)
- **เน้น:** พัฒนา Simulation (Python) และ Dashboard UI (Flutter)
- **เป้าหมาย:** สร้างตัวอย่างทางเทคนิคที่จับต้องได้ในผลงานวิชาการ

### 5. 📄 [LaTeX Publisher](.agents/skills/latex-publisher)
- **เน้น:** การจัดรูปแบบ LaTeX และ Markdown ขั้นสูง
- **เป้าหมาย:** ผลงานที่สวยงามระดับสากล พร้อมระบบบรรณานุกรมที่ถูกต้อง

### 6. 🔍 [Academic Reviewer Pro](.agents/skills/academic-reviewer-pro)
- **เน้น:** การตรวจประเมินคุณภาพภายใน (Internal Audit) ตามเกณฑ์ ก.พ.อ.
- **เป้าหมาย:** ลดความเสี่ยงในการถูกปฏิเสธ (Reject) และเพิ่มความลุ่มลึกของเนื้อหา

---

## 🚀 ขั้นตอนการผลิตผลงาน (Workflow)

1.  **Drafting:** ใช้ **Architect** หรือ **Writer** เพื่อเริ่มต้นโครงร่าง
2.  **Synthesis:** ใช้ **Agro-Physicist** และ **Coder** เพื่อเติมเนื้อหาเชิงลึกและหลักฐานทางเทคนิค
3.  **Auditing:** ใช้ **Reviewer Pro** เพื่อตรวจสอบคุณภาพตามเกณฑ์ ผศ./รศ./ศ.
4.  **Publishing:** ใช้ **Publisher** เพื่อสรุปไฟล์ต้นฉบับ (.tex / .pdf)
