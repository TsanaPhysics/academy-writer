# 🎓 Science Academic Writer: Modular AI Suite
**ชุดคำสั่ง AI อัจฉริยะเพื่อการผลิตผลงานวิชาการระดับสูง (มาตรฐาน รศ.)**

ยินดีต้อนรับสู่คู่มือเริ่มต้นใช้งาน **Science Academic Writer** ชุดเครื่องมือที่ช่วยให้อาจารย์และนักวิจัยสามารถร่างตำรา หนังสือ และเอกสารคำสอนได้อย่างเป็นระบบ รวดเร็ว และถูกต้องตามหลักวิชาการ

---

## 🛠️ โครงสร้างเครื่องมือ (Sub-Skills)
เราได้แยกความสามารถออกเป็น 4 ส่วน เพื่อให้ AI ทำงานได้อย่างแม่นยำที่สุด:

1.  **[Academic Textbook Architect](.agents/skills/academic-textbook-architect)**:
    - **หน้าที่:** วางโครงสร้างบทเรียน, ร่าง Outline, กำหนดหัวข้อย่อย (5-7 หัวข้อ), และคุมมาตรฐาน รศ.
    - **วิธีใช้:** ใช้เป็นลำดับแรกเพื่อสร้าง "กระดูกงู" ของหนังสือ

2.  **[Agro-Physicist](.agents/skills/agro-physicist)**:
    - **หน้าที่:** สังเคราะห์เนื้อหาฟิสิกส์เชิงลึก (กลศาสตร์, อุณหพลศาสตร์, ของไหล, แสง) และการประยุกต์ใช้ในงานเกษตรอัจฉริยะ (IoT/AI)
    - **วิธีใช้:** ใช้เมื่อต้องการเขียนเนื้อหาบรรยายที่ต้องมีสูตรและการวิเคราะห์เชิงฟิสิกส์

3.  **[Scientific Coder](.agents/skills/scientific-coder)**:
    - **หน้าที่:** เขียนโค้ดประกอบตำรา ทั้ง **Python** (Data Science/AI) และ **Flutter** (Mobile App/UI)
    - **วิธีใช้:** ใช้เมื่อต้องการสร้างตัวอย่างโค้ดหรือการจำลอง (Simulation) ในตำรา

4.  **[LaTeX Publisher](.agents/skills/latex-publisher)**:
    - **หน้าที่:** จัดรูปแบบไฟล์ LaTeX (.tex) และ Markdown (.md) อย่างเป็นระบบ พร้อมการจัดการบรรณานุกรม
    - **วิธีใช้:** ใช้ในขั้นตอนการจัดเก็บไฟล์และเตรียมต้นฉบับเพื่อส่งพิมพ์

## 🚀 ขั้นตอนการเริ่มต้นใช้งาน (Beginner Guide)

### 1. การนำไปใช้กับ AI (ChatGPT, Gemini, Claude, Grok)
คัดลอกเนื้อหาในไฟล์ `SKILL.md` ของแต่ละส่วนที่ต้องการใช้งาน ไปวางในช่องแชทแล้วแจ้ง AI ว่า:
> *"ช่วยจดจำคำสั่งและแนวทางในไฟล์นี้ เพื่อใช้เป็นมาตรฐานในการช่วยฉันร่างเนื้อหาตำราวิชาการ"*

### 2. ลำดับการทำงานที่แนะนำ (Recommended Workflow)
1.  **Step 1:** ส่ง "ชื่อเรื่อง" หรือ "คำอธิบายรายวิชา" ให้ **Architect** ร่างโครงสร้างทั้งหมด
2.  **Step 2:** ให้ **Agro-Physicist** เขียนเนื้อหาเชิงลึกในแต่ละบท (เป้าหมายบทละ 25-35 หน้า)
3.  **Step 3:** ให้ **Scientific Coder** สร้างโค้ด Python/Flutter ประกอบในหัวข้อที่เกี่ยวข้อง
4.  **Step 4:** ให้ **Publisher** จัดระบบไฟล์และสูตร LaTeX ให้สมบูรณ์

## 📁 รูปแบบการจัดเก็บไฟล์ที่แนะนำ
เพื่อให้โปรเจกต์ของอาจารย์ยั่งยืนและปรับปรุงง่าย ควรจัดเก็บดังนี้:
```text
[ชื่อหนังสือ]/
├── main.tex             # ไฟล์รวม LaTeX
├── chapters/            # แยกบท
│   ├── ch1_intro/       # บทที่ 1
│   │   ├── ch1.md, ch1.tex
│   │   ├── code/        # โค้ดประกอบ
│   │   └── images/      # รูปภาพ
└── references.bib       # บรรณานุกรม
```
