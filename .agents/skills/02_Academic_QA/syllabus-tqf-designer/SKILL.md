# Syllabus & TQF Designer

ทักษะในการออกแบบหลักสูตรรายวิชาที่ทันสมัย เน้นผลลัพธ์การเรียนรู้ และเป็นไปตามเกณฑ์ประกันคุณภาพการศึกษา

## Workflow
1. **CLO Mapping**: กำหนดผลลัพธ์การเรียนรู้ระดับรายวิชา (Course Learning Outcomes - CLOs) ให้สอดคล้องกับ PLOs
2. **Content Sequencing**: จัดลำดับเนื้อหาการสอนรายสัปดาห์ (15-16 สัปดาห์)
3. **TLM Selection**: เลือกวิธีการจัดการเรียนการสอน (Teaching & Learning Methods) เช่น Active Learning, Flipped Classroom
4. **Assessment Strategy**: ออกแบบวิธีการประเมินผลที่ตรงกับ CLOs (Constructive Alignment)
5. **Standard Compliance**: จัดทำเนื้อหาให้ครบถ้วนตามแบบฟอร์ม มคอ.3 (TQF3) หรือมาตรฐานสากล

## Triggers
- "ช่วยเขียน มคอ.3 สำหรับวิชา [ชื่อวิชา]"
- "ออกแบบ CLOs สำหรับรายวิชา [ชื่อหัวข้อ]"
- "แนะนำวิธีการสอนแบบ OBE สำหรับเนื้อหาเรื่อง [หัวข้อ]"

## Core Capabilities
- Constructive Alignment strategy.
- Learning taxonomies (Bloom, Fink).
- Quality assurance compliance (AUN-QA, ABET).

## Implementation Principles
- **Student-Centered**: มุ่งเน้นไปที่สิ่งที่นักเรียน "จะทำได้" หลังจบรายวิชา
- **Clarity**: ข้อกำหนดและเกณฑ์การให้คะแนนต้องโปร่งใสและชัดเจน
- **Future-Ready**: บูรณาการทักษะศตวรรษที่ 21 และเทคโนโลยีใหม่ๆ ในเนื้อหา
