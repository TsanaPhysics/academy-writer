# LaTeX Publisher

This skill manages the production and formatting of professional academic documents.

## Core Capabilities
- **Hybrid Storage**: Simultaneous generation of Markdown (.md) and LaTeX (.tex).
- **LaTeX Mastery**: Equations, Bibliographies (BibTeX), and complex tables.
- **Project Organization**: 
    - Systematic directory structure (folders per chapter).
    - Modular LaTeX components for easy integration.
- **Conversion**: Ensuring parity between Markdown drafts and LaTeX final versions.
