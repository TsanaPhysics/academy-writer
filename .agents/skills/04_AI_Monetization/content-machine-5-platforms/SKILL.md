# Content Machine 5 Platforms

ทักษะนี้ช่วยเปลี่ยนหัวข้อหรือเนื้อหาหลักเพียงหนึ่งเดียวให้กลายเป็นคอนเทนต์ที่พร้อมใช้งานใน 5 แพลตฟอร์มหลัก เพื่อประสิทธิภาพสูงสุดในการสื่อสาร

## Workflow
1. **Input Analysis**: รับหัวข้อ (Topic), เนื้อหาหลัก (Context), หรือลิงก์ที่ต้องการสรุป
2. **Platform Adaptation**: ปรับแต่งโทนและรูปแบบเนื้อหาให้เข้ากับธรรมชาติของแต่ละแพลตฟอร์ม:
   - **Facebook**: เน้นการเล่าเรื่อง (Storytelling) และ Engagement
   - **Instagram**: เน้นคำบรรยายที่สั้น กระชับ และไอเดียสำหรับภาพ/วิดีโอ
   - **TikTok**: เน้น Hook ที่น่าสนใจ และสคริปต์สั้นๆ สำหรับวิดีโอ
   - **X (Twitter)**: เน้นความกระชับ ทันเหตุการณ์ และ Thread ที่น่าติดตาม
   - **Newsletter**: เน้นเนื้อหาเชิงลึก ข้อมูลที่เป็นประโยชน์ และ Call to Action
3. **Hashtag Generation**: เลือก Hashtag ที่เกี่ยวข้องและกำลังเป็นกระแส
4. **Final Review**: ตรวจสอบความถูกต้องของภาษาและโทนเสียงให้สอดคล้องกัน

## Core Capabilities
- Multi-platform copywriting.
- Hook generation for short-form video.
- Newsletter structuring.
- Strategic hashtag selection.

## Implementation Principles
- **One Topic, Multi-Format**: รักษาใจความสำคัญเดิมแต่เปลี่ยนรูปแบบการนำเสนอ
- **Platform Native**: ใช้ภาษาและสัญลักษณ์ (Emoji) ที่เหมาะสมกับแต่ละกลุ่มเป้าหมาย
- **Call to Action (CTA)**: ต้องมี CTA ที่ชัดเจนในทุกโพสต์
