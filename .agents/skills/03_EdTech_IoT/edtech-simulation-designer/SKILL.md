# EdTech Simulation Designer

ทักษะในการเปลี่ยนบทเรียนวิชาการที่ซับซ้อน (เช่น ฟิสิกส์ หรือ เคมี) ให้กลายเป็นสื่อการสอนที่นักเรียนสามารถโต้ตอบและทดลองได้แบบเรียลไทม์

## Workflow
1. **Physics/Scientific Concept Mapping**: วิเคราะห์หลักการทางวิทยาศาสตร์ที่ต้องการนำเสนอ (เช่น กฎของนิวตัน, การเคลื่อนที่)
2. **Logic Design**: ออกแบบอัลกอริทึมการคำนวณที่สอดคล้องกับหลักความเป็นจริง
3. **Frontend Integration**: ใช้ p5.js หรือ Three.js ในการสร้างภาพกราฟิกแบบ 2D/3D ที่โต้ตอบได้
4. **AI-Driven Feedback**: บูรณาการ AI เพื่อคอยแนะนำหรืออธิบายสิ่งที่เกิดขึ้นใน Simulation ขณะที่นักเรียนกำลังทดลอง
5. **UI/UX Optimization**: ออกแบบหน้าจอให้ใช้งานง่าย (User-friendly) และรองรับการใช้งานบนมือถือ

## Triggers
- "ช่วยออกแบบ Simulation เรื่อง [หัวข้อ]"
- "เขียนโค้ด p5.js สำหรับการทดลอง [ชื่อการทดลอง]"
- "ปรับปรุงระบบตอบโต้อัตโนมัติในสื่อการสอนชิ้นนี้"

## Core Capabilities
- Scientific logic implementation in JavaScript.
- 2D/3D visual simulation (p5.js, Three.js).
- Interactive learning experience design.

## Implementation Principles
- **Accuracy**: ข้อมูลและการแสดงผลต้องมีความถูกต้องตามหลักการทางวิทยาศาสตร์
- **Interactivity**: เน้นให้ผู้เรียนได้ลองปรับเปลี่ยนตัวแปร (Variables) และเห็นผลลัพธ์ทันที
- **Clarity**: ใช้ภาพประกอบที่เรียบง่ายแต่ทรงพลังในการสื่อสารแนวคิด
