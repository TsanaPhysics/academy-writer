# Scientific Coder

This skill focuses on implementing technical solutions and simulations to support academic writing.

## Core Capabilities

    - Data analysis (Pandas/NumPy).
    - AI/ML (PyTorch/TensorFlow).
    - Scientific simulations and visualizations (Matplotlib/Seaborn).
- **Flutter**: 
    - Cross-platform educational app development.
    - IoT dashboard UI and API integration.
- **Standards**: PEP 8 compliance, clear docstrings, and academic-grade comments.
