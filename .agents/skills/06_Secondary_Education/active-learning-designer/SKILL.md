# Active Learning Activity Designer

ทักษะในการเปลี่ยนห้องเรียนแบบบรรยายให้กลายเป็นพื้นที่แห่งการเรียนรู้ที่สนุกสนานและมีความหมาย

## Workflow
1. **Context Analysis**: รับหัวข้อและวิเคราะห์ปัญหาหรือสิ่งที่นักเรียนมักจะสับสน
2. **Method Selection**: เลือกรูปแบบกิจกรรมที่เหมาะสม (เช่น Case Study, Role Play, Debate, หรือ Simulation)
3. **Activity Scaffolding**: ออกแบบขั้นตอนการดำเนินกิจกรรมที่ชัดเจน (Introduction -> Action -> Reflection)
4. **Gamification Elements**: เพิ่มองค์ประกอบของเกม (คะแนน, เลเวล, ความท้าทาย) เพื่อกระตุ้นแรงจูงใจ
5. **Debriefing Strategy**: เตรียมคำถามสำหรับการสะท้อนคิด (Reflection) หลังจบกิจกรรม

## Triggers
- "คิดเกมประกอบการสอนเรื่อง [หัวข้อ]"
- "ออกแบบโปรเจกต์ (PBL) สำหรับวิชา [ชื่อวิชา]"
- "หาวิธีสอนเรื่องยากๆ ให้เข้าใจง่ายด้วยกิจกรรม"

## Core Capabilities
- Gamification in education.
- Project-Based Learning (PBL) design.
- Experiential learning structure.

## Implementation Principles
- **Engagement**: นักเรียนต้องเป็นผู้เล่นหลักในกิจกรรม
- **Safe to Fail**: พื้นที่กิจกรรมควรอนุญาตให้ลองผิดลองถูกได้
- **Real-world Connection**: เชื่อมโยงสิ่งที่เรียนกับชีวิตจริงหรืออาชีพในอนาคต
