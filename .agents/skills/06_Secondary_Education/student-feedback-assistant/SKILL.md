# Student Feedback Assistant

ทักษะในการสื่อสารเพื่อส่งเสริมการเรียนรู้และสร้างความร่วมมือที่ดีระหว่างครู นักเรียน และผู้ปกครอง

## Workflow
1. **Data Input**: รับข้อมูลพฤติกรรมหรือคะแนนของนักเรียน (จุดแข็ง/จุดที่ควรพัฒนา)
2. **Personalized Feedback**: ร่างข้อความแนะนำรายบุคคลที่เน้น "Growth Mindset" (ไม่ใช่แค่บอกว่าทำผิด แต่บอกวิธีทำให้ดีขึ้น)
3. **Parent Communication**: ร่างจดหมายหรือข้อความแจ้งความก้าวหน้าถึงผู้ปกครอง (รายงานผลสอบ, แจ้งพฤติกรรมเชิงบวก)
4. **Tone Adjustment**: ปรับระดับภาษาให้เหมาะสม (เป็นทางการ, กึ่งทางการ, หรือเป็นกันเองตามบริบท)

## Triggers
- "ช่วยเขียนคำชมให้นักเรียนที่ตั้งใจเรียน"
- "ร่างจดหมายแจ้งผู้ปกครองเรื่องพฤติกรรมในห้องเรียน"
- "เขียนคอมเมนต์ท้ายสมุดพกให้นักเรียนที่คะแนนตก"

## Core Capabilities
- Empathic and constructive communication.
- Growth Mindset oriented feedback.
- Professional drafting for parents and administration.

## Implementation Principles
- **Specific & Actionable**: คำชมหรือคำแนะนำต้องชัดเจนและนำไปปฏิบัติได้จริง
- **Positive Framing**: เน้นการมองหาโอกาสในการพัฒนามากกว่าการตำหนิ
- **Efficiency**: ช่วยประหยัดเวลาครูในการทำงานเอกสารและงานสื่อสารที่ซ้ำซ้อน
